# Eadria Wiki Agent Instructions

This repository is the public GitHub Pages site for Eadria.

## Scope

- Publish only public-facing lore and site files here.
- Do not add private drafts, API keys, tokens, credentials, or private agent configuration.
- Keep links relative so the site works at `/eadria/` on GitHub Pages.

## Style

- Tone: mythic, historical, geographic, and wiki-like.
- Prefer clear canon over speculation.
- Do not invent major canon without explicit approval.
- Mark uncertain or placeholder content clearly.

## Site conventions

- `index.html` is the home page.
- Region pages live in `regions/`.
- People/culture pages live in `peoples/`.
- Public structured data lives in `data/`.
- Public images and maps should live in `assets/images/`.
- Add new public pages to the relevant index and navigation where appropriate.

## GitHub Pages

This site is intended to publish from the `main` branch root folder with GitHub Pages.
